Throughout my professional career, I started as a designer in Photoshop and then transitioned to programming by working on a game server for some friends. Initially, as with any beginner, I found learning challenging, but my interest and enthusiasm for technology only grew. I then decided to learn my first programming language, PHP, and dedicated myself to continuous courses, gaining knowledge about networks and web servers. I also learned HTML, CSS, and Bootstrap, and since there were few programmers on the game server, I leveraged my skills to program for other similar servers and was recommended among game owners. I gained significant popularity in these servers, which were for the game GunZ: The Duel. Here, I also delved into C# and C++, the languages used by the game to run on Windows. Additionally, I focused on 3D modeling, and you can find some of the 3D models I created for the game in Blender in my portfolio. I emphasize this game a lot because it was the catalyst that got me started in programming, design, 3D modeling, and cybersecurity. As it was a community with a low demand for developers, I easily made my way through it with all the knowledge I acquired, helping my friends to have the best server possible.
